# **MiniScanner**  
*Антивирусный сканер, основанный на системе плагинов.*  
Плагины взаимодействуют с системой и основой программы через специальный **API**.  
Также плагины могут реализовывать свои **API** для взаимодействия с системой.

---

## **Текущие версии различных составляющих**

### **Base: v1.2.0**  
**Основные компоненты MiniScanner, исключая API**

### **API: v1.1.0**  
**API для взаимодействия с системой**

#### **LNK Tools: v1.0.0**  
*Взаимодействие с ярлыками, к примеру для их анализа*

#### **ProgramList: v1.0.0**  
*Получение списка установленных приложений*

#### **Autorun Utils: v1.0.0**  
*Получение информации об автозапуске*

#### **Paths: v1.0.1**  
*Обычные пути системы*
